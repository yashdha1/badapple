# badapple

video and audio player. 


## Run

```powershell
uv run python main.py
```

## Download a Video

```powershell
uv run python main.py --add "<video-link>"
```

## Controls

- `Space` / `P`: pause or play
- `V`: next ASCII mode
- `T`: next theme
- `F`: fullscreen video-only
- `H`: toggle HUD
- `Q` / `Esc`: quit
